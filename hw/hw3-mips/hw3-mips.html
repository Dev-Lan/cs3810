Homework 3: MIPS Assembly

This assignment will teach you to write simple MIPS assembly programs. We will be using SPIM simulator, a small simulator that implements the MIPS architecture we covered in class.

NOTE: YOU CANNOT PUBLICLY RELEASE SOLUTIONS TO THIS HOMEWORK. It's ok to show your work to your future employer as a private github/gitlab repo, however any public release is prohibited.

Install SPIM

As the first step of this homework install SPIM simulator. Visit download SPIM and choose the most recent version for the operating systme you're running.

While SPIM is pretty much intuitive and self-explanatory, you might want to watch a short introduction video that expains how to use it How to Program in MIPS! (QTSpim) (Beginner)

You can use any editor to develop MIPS code (after all to run it in SPIM you will simply open a file), but I recommend Visual Studio Code as it is simple, runs on all platforms, and supports MIPS syntax with a plugin.

Count set bits from 0 to N

Now you're ready to work on the homework itself. Your assignment is to implement a MIPS program that counts all set bits (the ones that are 1) in numbers from 1 to N (including N). Specifically, your program asks a user to input an integer number and then outputs a number of set bits. For example, if the user enters 5, then if you look at binary representation of all the numbers from 0 to 5 and their bits: 
0b0 0b1 0b10 0b11 0b100 0b101
The sum of all set bits is 7.
